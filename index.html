Angularify

Is a boilerplate for developing, testing and deploying AngularJS applications with ES6 (ECMAScript 6) using Browserify bundler.

Technologies:

Getting Started

$ git clone https://github.com/yabasha/angularify.git
$ cd angularify
$ npm install

You need Gulp installed globally:

$ npm i -g gulp

Run

Development Server

$ gulp dev

Build the project

$ gulp production

Deploy

If you want to deploy your app under subfolder, don't forget to configure 
<base href="/subfolder/">
or leave it 
<base href="/">

if you want to install it under the root folder.
